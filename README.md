# ✨ AI Text & Image Generator 📱  
**Unlock creativity on the go — powered by ChatGPT API**

## 🚀 Overview  
**AI Text & Image Generator** is a user-friendly Android application that transforms your ideas into stunning visuals and compelling text using the powerful ChatGPT API. Whether you’re a creator, designer, or just curious — this app is your creative companion.

- 🎨 Generate AI-powered images from text prompts  
- 📝 Create text content — stories, bios, brainstorms, tweets, and more  
- ⚡ Smooth and intuitive UI designed for all users  
- 🔮 Fast, real-time results backed by OpenAI’s ChatGPT  

## 🌟 Features  
- **Text Generation**: Instantly produce creative or informative content  
- **Image Generation**: Turn descriptions into AI-crafted visuals  
- **Modern UI**: Clean and easy-to-use interface with dark mode support  
- **ChatGPT Integration**: Reliable, intelligent outputs in seconds  

## 🛠️ Built With  
- Android (Kotlin/Java)  
- OpenAI ChatGPT API
- Stability AI API
- Jetpack Components  
- Material Design  

## 📦 Installation  
1. Clone this repository:
    ```bash
    git clone https://github.com/yourusername/ai-text-image-generator.git
    ```
2. Open the project in **Android Studio**
3. Add your OpenAI API key in `local.properties` or `BuildConfig`:
    ```properties
    OPENAI_API_KEY=your_api_key_here
    ```
4. Build and run on your emulator or Android device

## 🔐 Requirements  
- Android 7.0 (API level 24) or above  
- Internet connection

## 💡 How to Use  
1. Type your prompt in the input field  
2. Choose “Text” or “Image” generation  
3. Tap **Generate**  
4. Your content is ready to use or share! 🎉
